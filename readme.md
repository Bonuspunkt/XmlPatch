# XmlPatch
.net implementation of [rfc5261](http://tools.ietf.org/html/rfc5261)

tested with [Appendix A](http://tools.ietf.org/html/rfc5261#appendix-A)

nearly complete, only 'Several Patches With Namespace Mangling' fails
